## How to execute

1. clone github's repository
    ```bash
    git clone git@github.com:youtame/kitasenzoku.git
    ```

2. build docker container
    ```bash
    cd kitasenzoku
    docker compose up -d --build
    ```

3. visit 102do site
    [http://localhost:5173](http://localhost:5173 "http://localhost:5173")
